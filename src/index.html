The Autumn Architecture

Autumn | Abstract

Autumn is a modular, portable, end-to-end Platform engineered for developing performant Machine Intelligence applications. It consits currently of two Frameworks Collenchyma and Leaf.


Leaf

Leaf is an imperative, expressive, cross-platform Machine Intelligence Framework. Its clear architecture keeps it maintainable, easy-to-use and extendable e.g. with the latest research for cutting-edge-performance.


Collenchyma

Collenchyma is a portable, platform-agnostic, high-performance computation Framework. Applications build with Collenchyma can be deployed and executed in parallel on one or many hetereogenous devices on any host machine.

Autumn Features


Build maintainable Machine Intelligence applications, that can be deployed into the cloud for production, executed for testing on your desktop and shipped to smartphones, IoT and embedded devices. No code changes required.

PERFORMANT

Autumn enables developers to quickly build maintainable applications, train the models and efficiently execute them in production, reducing development and maintainance costs and increasing productivity.

PRACTICAL

Autumn enables developers to understand the whole stack, write maintainable plugins and safely implement the latest research outcomes into their applications.

PORTABLE

Autumn gives you the choice to run, deploy and test your Machine Intelligence applications whereever you want. Any machine, any OS, any hardware.
